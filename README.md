# Запуска проекта
---
### Подготовка к запуску:
1) Установить PostgresSQL и СУБД
2) Установить NodeJS v24+
3) В проекте создать файл .env и добавить в него переменные:
    * DATABASE_URL - Адрес по которому будет осуществляться подключение к БД
    * DB_HOST - Хостнейм на котором будет работать БД
    * DB_PORT - Порт на кототором будет работать БД
    * DB_USER - Имя пользователя в БД
    * DB_PASS - Пароль для доступа к БД
    * DB_NAME - Имя ДБ
    * PORT - Порт на котором будет работать сервер
---
### Запуск проекта

1)  > #### Установка зависимостей 
    > ```npm i```
2)  > #### Генерация файлов для работы Prisma 
    > ```npx prisma generate```
3)  > #### Миграция схем Prisma в вашу БД 
    > ```npx prisma migrate```
